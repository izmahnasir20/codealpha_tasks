# CodeAlpha Task 3 – Personal Portfolio Website

## Description
This project is a personal Portfolio Website developed as part of the CodeAlpha Front-End Development internship.  
It showcases the developer's skills, projects, resume, and contact information in a clean and responsive layout. The portfolio includes smooth scrolling animations, hover effects, and optionally JavaScript enhancements for interactivity.  

## Technologies Used
- HTML
- CSS
- JavaScript 

## Features
- About Me section to highlight skills and background
- Projects section showcasing completed projects
- Resume download option
- Contact section with a functional form or contact info
- Smooth scroll animations and hover effects
- Fully responsive design for desktops, tablets, and mobile devices
- Bonus: Deployed on GitHub Pages or Netlify

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Navigate through sections using menu links or scrolling.
