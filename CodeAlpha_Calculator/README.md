# CodeAlpha Task 2 – Calculator

## Description
This project is a basic Calculator built as part of the CodeAlpha Front-End Development internship.  
It performs all standard arithmetic operations (addition, subtraction, multiplication, division) and provides a user-friendly interface for real-time calculations. The project also includes clear screen functionality and interactive buttons for user input.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Addition, Subtraction, Multiplication, Division
- User-friendly interface with clickable buttons
- Real-time calculation display
- Clear screen functionality
- Bonus: Keyboard support and styling enhancements

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Use the buttons or keyboard to perform calculations.
