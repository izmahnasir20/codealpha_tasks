# CodeAlpha Task 1 – Image Gallery

## Description
This project is an Image Gallery developed as part of the CodeAlpha internship.  
It allows users to browse images using next/previous navigation buttons and view images in a lightbox. The gallery includes hover effects, smooth transitions, and is fully responsive across different screen sizes.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Next/Previous navigation buttons
- Lightbox view for enlarged images
- Hover effects and smooth CSS transitions
- Fully responsive design for desktops, tablets, and mobile devices
- Bonus: Image filters and category selection 

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Navigate through the gallery using the buttons or click on images for lightbox view.
