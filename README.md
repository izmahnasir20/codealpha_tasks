# CodeAlpha Internship Tasks – Front-End Development

This repository contains all the tasks completed during my Front-End Development internship at CodeAlpha.  
Each task is organized in its own folder with source code, documentation, and instructions for running.

---

## Table of Contents

| Task Number | Task Name | Folder | Description |
|------------|-----------|--------|-------------|
| Task 1 | Image Gallery | [CodeAlpha_image_gallery](CodeAlpha_image_gallery) | An interactive image gallery with next/previous navigation and lightbox view. Includes hover effects, smooth transitions, and responsive design. |
| Task 2 | Calculator | [CodeAlpha_Calculator](CodeAlpha_Calculator) | A basic calculator performing addition, subtraction, multiplication, and division. Features real-time result display, clear screen, and optional keyboard support. |
| Task 3 | Personal Portfolio Website | [CodeAlpha_Portfolio](CodeAlpha_Portfolio) | A personal portfolio showcasing skills, projects, resume, and contact info. Includes smooth scroll animations, hover effects, and responsive design. Optional deployment on GitHub Pages or Netlify. |
| Task 4 | Music Player | [CodeAlpha_Music_Player](CodeAlpha_Music_Player) | A music player with play, pause, next, previous controls, progress bar, volume control, and playlist support. Bonus: Auto-play functionality for continuous playback. |
