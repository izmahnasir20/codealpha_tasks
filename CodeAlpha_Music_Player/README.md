# CodeAlpha Task 4 – Music Player

## Description
This project is a Music Player built as part of the CodeAlpha Front-End Development internship.  
It allows users to play, pause, skip songs, control volume, and view song details including title, artist, and duration.  
The player features a progress bar, playlist functionality, and optional auto-play for the next song.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Play, Pause, Next, and Previous song controls
- Display song title, artist, and duration
- Progress bar to show current song position
- Volume control
- Playlist feature to select and play songs
- Bonus: Auto-play next song after the current one ends
- Responsive and interactive design

## How to Run
1. Clone or download the repository.
2. Navigate to the project directory.
3. Open `index.html` in any modern web browser.
4. Use the buttons or playlist to play songs.

## Usage Instructions
- Click "Play" to start the song.
- Click "Pause" to stop playback.
- Use "Next" and "Previous" to navigate songs.
- Drag the progress bar to jump to a different part of the song.
- Adjust the volume using the slider.
- Select songs from the playlist.
- Auto-play will automatically play the next song.
